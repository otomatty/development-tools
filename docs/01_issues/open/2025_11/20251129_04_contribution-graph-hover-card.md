# Issue: コントリビューショングラフのホバーカード実装

## 概要

コントリビューショングラフの各日付セルにホバーした際、詳細情報をリッチなホバーカードとして表示する。

## 現状

- `src/components/home/contribution_graph.rs` で `title` 属性によるシンプルなツールチップを表示
- 表示内容: `{日付}: {コントリビューション数} contributions`
- スタイリングはブラウザデフォルト

## 要件

### 機能要件

1. **カスタムホバーカードの表示**

   - マウスホバー時にカスタムデザインのカードを表示
   - セルの近くに表示（画面端では位置調整）

2. **表示情報**

   - 日付（曜日付き）: 例「2025 年 11 月 29 日 (金)」
   - コントリビューション数: 例「5 contributions」
   - コントリビューションレベル: 「Low / Medium / High / Very High」
   - （将来拡張）活動種別の内訳（コミット数、PR 数等）

3. **アニメーション**
   - フェードイン/アウトアニメーション
   - アニメーション無効設定時はアニメーションなし

### 技術要件

1. **HoverCard コンポーネント**

   - 再利用可能なホバーカードコンポーネントを作成
   - position: absolute でセル位置に基づいて配置
   - z-index で他要素の上に表示

2. **ホバー状態管理**

   - マウス enter/leave イベントでホバー状態を管理
   - ホバー中のセル情報を state で保持

3. **位置計算**
   - セルの Bounding Rect を取得
   - 画面端での位置調整（右端なら左に表示等）

## 影響範囲

### 新規ファイル

- `src/components/home/hover_card.rs` - 汎用ホバーカードコンポーネント

### 修正ファイル

- `src/components/home/mod.rs` - hover_card モジュールの公開
- `src/components/home/contribution_graph.rs` - ホバーカード統合

## UI/UX デザイン

```
     ┌─────────────────────────┐
     │ 📅 2025年11月29日 (金)   │
     │ ─────────────────────── │
     │ 🟩 5 contributions       │
     │ Level: High             │
     └─────────────────────────┘
            ▼
         [セル]
```

### カラースキーム

- 背景: `bg-gm-bg-card/95 backdrop-blur-sm`
- ボーダー: `border-gm-accent-cyan/30`
- テキスト:
  - 日付: `text-white font-bold`
  - コントリビューション数: `text-gm-success`
  - レベル: コントリビューションレベルに応じた色

## テストケース

1. **TC-001**: セルにホバーするとホバーカードが表示される
2. **TC-002**: ホバーを外すとホバーカードが非表示になる
3. **TC-003**: 日付、コントリビューション数、レベルが正しく表示される
4. **TC-004**: 画面右端のセルでホバーカードが左側に表示される
5. **TC-005**: アニメーション無効時にアニメーションなしで表示される
6. **TC-006**: コントリビューション 0 の日でも正しく表示される
